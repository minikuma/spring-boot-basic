### Configuration Classes
---    
스프링 부트는 자바 코드 기반 설정을 지원하고 해당 방법을 권장하고 있다. 참고로 과거 버전은 XML 설정으로 지원하였다. ```@Configuration```을 지원하고 있고 해당 애노테이션을 붙힌 자바 코드는 설정으로 간주한다. 설정은 아래 3가지 방법이 대표적이다.       

1. 직접 자바 코드 만들고 애노테이션 붙이기
* ```@Configuration```
* ```@Import``` 다른 클래스의 설정도 가져올 수 있다.
2. 모든 설정을 스프링 부트에게 맡긴다. ```@ComponentScan```.
